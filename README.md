## 🪄 제출 방식
- 본인 이름으로 브랜치를 만든다.
- 주차별 디렉토리/문제번호 디렉토리/문제번호_본인 이름.cpp 으로 디렉토리를 만들고 코드를 올린다.
- ex) 파일 네이밍: `week1/BOJ_1234/BOJ_1234_ohji.cpp`
- PR을 올린다. (남기고 싶은 내용을 남긴다.)
- ex) PR 네이밍: `[BOJ 16985] Maaaaaaaaaze - 오지현`
